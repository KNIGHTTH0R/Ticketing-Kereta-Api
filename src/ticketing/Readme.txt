Cara Installasi dan Menjalankan Aplikasi Tiketing Kereta Api

1. Install Xampp v3.2.1 terlebih dahulu lalu jalankan Sql dan Apache
2. Buka web browser seperti mozilla atau chrome kemudian ketik http://localhost/phpmyadmin/
3. Import file tubespbo.sql sebagai database dalam phpmyadminnya
4. Lalu jalankan aplikasi dengan cara membuka Netbeans dan open project CobaCoba.
5. kemudian run project Cobacoba

Selesai.
