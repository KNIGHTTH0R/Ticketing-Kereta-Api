Cara Installasi dan Menjalankan Aplikasi Tiketing Kereta Api

1. Install Xampp lalu dia menjalankan Sql dan Apache
2. Buka web browser http://localhost/phpmyadmin/
3. Import file tubespbo.sql sebagai database dalam phpmyadminnya
4. Lalu jalan kan aplikasi dengan cara menjalankan Netbean dahulu lalu open project CobaCoba.

Selesai.
