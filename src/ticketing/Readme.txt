Cara Installasi dan Menjalankan Aplikasi Tiketing Kereta Api

1. Install Xampp v3.2.1 terlebih dahulu lalu jalankan Sql dan Apache
2. Buka web browser seperti mozilla atau chrome kemudian ketik http://localhost/phpmyadmin/
3. Import file tubespbo.sql sebagai database dalam phpmyadminnya
4. Lalu jalankan aplikasi dengan cara membuka Netbeans dan open project CobaCoba.
5. kemudian run project Cobacoba

link download xampp :
https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/1.8.3/xampp-win32-1.8.3-5-VC11-installer.exe/download

Deskripsi Umum Aplikasi :
  Aplikasi ini dapat digunakan oleh petugas stasiun. Aplikasi ini akan mengatasi proses pencetakan tiket kereta dimulai dari nama kereta, gerbong kereta, stasiun yang dilewati, masinis yang bertugas, rute perjalanan kereta, waktu keberangkatan dari kota asal, waktu kedatangan di kota tujuan, hingga menampilkan informasi mengenai jumlah tempat duduk yang masih tersedia atau tidak kepada penumpang/pembeli. 
  Apabila petugas yang menggunakan aplikasi ini, maka petugas dapat melihat, mengubah, menghapus, ataupun menambahkan data tiket kereta api. Namun, penumpang/pembeli tiket hanya dapat memesan tiket keretanya saja. 

Selesai.
